# Thoughts Mapper

A free, local-first visual thinking app inspired by associative knowledge tools.

## Run

Open `index.html` in a modern browser.

## Current MVP

- Visual thought graph with automatic focus layout
- Active thought centering with softer inactive context
- Rounded thought cards with animated active focus rings
- Add thoughts from the sidebar
- Select, edit, and delete thoughts
- Notes for each thought
- Connect thoughts together as parent/child relationships
- Layered focus layout: ancestors move above, the active thought stays central, children and descendants flow into readable rows, and siblings stay visible in a peer row
- Smooth animated focus transitions when selecting a thought
- Search by title, type, and notes
- Quick capture inbox for thoughts that are not linked yet
- Tags and tag filtering
- Backlinks and `[[mention]]` link suggestions
- Markdown notes with checklist rendering
- Parent, child, and related links
- Undo and redo for map edits
- Fit, center, reset, pan, and zoom controls
- Light and dark visual modes
- Non-blocking Settings panel for line thickness, connection type, line endpoint style, and colour scheme
- Pastel and animated background presets: fire, leaves, black hole, aurora, rain, snow, nebula, and ocean
- Right-click a thought to create a connected parent or child thought
- Browser-local persistence with IndexedDB and localStorage fallback
- JSON export and import
- Markdown export
